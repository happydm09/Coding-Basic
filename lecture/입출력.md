# 입/출력
- 입력과 출력

## 입력(Input)
- (사용자로부터) 값을 받는 것 <br>


문법: <br>
```python
input()
```

입력 받은 값을 저장하고 싶다면 변수에다가 저장해주면 된다.
```python
a = input()
```

## 출력(Output)
- 값을 내보내는 것 <br>

문법: <br>
```python
print(값)
```

예)
```python
print("Hello!") # "Hello!" 가 출력 될거임
```
