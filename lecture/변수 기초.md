# 변수 기초
- 변할 수 있는 수

## 문법

(변수명) = (값)

예)
```python
a = 1
```

<br>
'='은 대입 연산자로서 수학적 의미의 같다는 뜻이 아님 <br>
쉽게 말해 '이 변수는 앞으로 이 값을 가진다'는 뜻 <br>

<br>

예) <br>
```python
a = 1
a = 2
```
설명: 'a = 1'은 'a는 1이 된다'는 뜻. <br>
다음 줄 'a = 2'는 a의 값이 1에서 2로 바뀐 것. ('1 = 2' 가 아님) <br>
즉, a는 앞으로 2가 됨.
